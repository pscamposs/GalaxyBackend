## Sobre

O "Minecraft Store Backend" é a infraestrutura do servidor para um e-commerce dedicado à venda de produtos relacionados ao universo do jogo Minecraft. Este projeto foi desenvolvido utilizando Express.js, Node.js, TypeScript e MongoDB para oferecer uma plataforma para gerenciar produtos, pedidos e usuários do e-commerce.

## Tecnologias Utilizadas

- Node.js
- Express.js
- TypeScript
- MongoDB

## Como Usar

1. Certifique-se de ter o Node.js e o MongoDB instalados em sua máquina.
2. Clone este repositório usando `git clone https://github.com/pscamposs/GalaxyBackend.git`.
3. No terminal, navegue até o diretório do projeto.
4. Instale as dependências usando `npm install`.
5. Configure as variáveis de ambiente necessárias para conexão com o banco de dados e outras configurações específicas do ambiente.
6. Execute o servidor usando `npm start`.
